GhostNote 隐私政策

最后更新日期：2026年2月6日

核心承诺：GhostNote 是一款 100% 本地运行的智能笔记应用。我们不收集、不传输、不存储任何您的个人数据到远程服务器。您的所有数据仅存储在您的设备上，您拥有完全的控制权。

1. 数据收集与使用

1.1 我们收集的信息

GhostNote 在您的设备本地存储以下信息：

数据类型 用途 存储位置
笔记内容 保存您创建的文字和语音笔记 设备本地数据库
音频录音 保存语音笔记的原始音频 设备本地文件
位置信息（城市级别） 为笔记添加地理上下文（可选） 设备本地数据库
日历事件 关联提醒到系统日历（可选） 系统日历应用
音乐播放信息 记录创建笔记时播放的音乐（可选） 设备本地数据库
使用偏好设置 保存语言选择等应用设置 设备本地存储

1.2 我们不收集的信息

  • 个人身份信息（姓名、电话、邮箱地址）
  • 精确位置数据（GPS 坐标）
  • 设备标识符（IDFA、广告ID）
  • 使用分析数据
  • 崩溃报告
  • 任何可用于追踪您的信息

2. 权限说明

GhostNote 可能请求以下系统权限，所有权限仅用于本地功能：

权限 用途 必需性
麦克风 录制语音笔记 必需（核心功能）
语音识别 将语音转换为文字（本地处理） 必需（核心功能）
位置服务 添加城市级别的地理上下文 可选
日历 将提醒添加到系统日历 可选
媒体资料库 记录当前播放的音乐名称 可选

重要说明：所有可选权限均可拒绝，不会影响应用的核心笔记功能。

3. 数据存储与安全

3.1 本地存储

  • 所有数据仅存储在您的 iOS 设备本地
  • 使用 SQLite 数据库存储笔记内容和元数据
  • 音频文件存储在应用沙盒的 Documents 目录
  • 不使用 iCloud 或任何云服务同步数据

3.2 数据安全

  • 数据受到 iOS 系统级别的加密保护
  • 应用沙盒机制确保其他应用无法访问您的数据
  • 不与任何第三方共享数据
  • 不通过网络传输任何用户数据

4. 第三方服务

GhostNote 不使用任何第三方服务，包括但不限于：

  • 分析服务（如 Google Analytics、Firebase Analytics）
  • 广告网络
  • 崩溃报告服务
  • 云存储服务
  • 社交媒体 SDK

唯一使用的外部库是开源的 GRDB.swift，仅用于本地数据库管理，不涉及任何网络通信。

5. AI 功能与隐私

GhostNote 使用以下 AI 功能，所有处理均在设备本地完成：

  • 语音识别：使用 Apple Speech Recognition API，完全离线处理
  • 自然语言处理：使用 Apple Natural Language 框架进行意图分类和情感分析
  • 智能分类：使用本地机器学习模型识别笔记类型（提醒、待办、想法）
  • 语义搜索：使用 Apple NLEmbedding 生成语义向量，在本地进行相似性搜索

隐私保证：您的语音和文字内容不会发送到任何远程服务器进行处理。

6. 数据保留与删除

  • 您可以随时在应用内删除任何笔记
  • 删除的数据将从本地数据库中永久删除
  • 卸载应用将删除所有应用数据，包括笔记、录音和设置
  • 我们不保留任何数据的备份副本

7. 儿童隐私

GhostNote 适合所有年龄段的用户使用。由于我们不收集任何个人信息，因此不会有意收集 13 岁以下儿童的个人数据。

8. 隐私政策的更新

我们可能会不定期更新本隐私政策。更新后，我们会修改页面顶部的"最后更新日期"。重大变更会在应用内通知您。

9. 您的权利

由于所有数据仅存储在您的设备上，您拥有完全的控制权：

  • 访问权：您可以随时查看所有笔记数据
  • 修改权：您可以随时编辑或删除笔记
  • 删除权：您可以删除单条笔记或卸载应用删除所有数据
  • 撤回权限：您可以在 iOS 设置中随时撤回已授予的权限

10. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

GhostNote Privacy Policy

Last Updated: February 6, 2026

Core Commitment: GhostNote is a 100% local-first intelligent note-taking app. We do not collect, transmit, or store any of your personal data to remote servers. All your data is stored only on your device, and you have complete control.

1. Data Collection and Use

1.1 Information We Collect

GhostNote stores the following information locally on your device:

Data Type Purpose Storage Location
Note Content Save your text and voice notes Local device database
Audio Recordings Store raw audio of voice notes Local device files
Location (City-level) Add geographic context to notes (optional) Local device database
Calendar Events Link reminders to system calendar (optional) System Calendar app
Music Playback Info Record music playing when creating notes (optional) Local device database
User Preferences Save app settings like language choice Local device storage

1.2 Information We Do NOT Collect

  • Personal identification information (name, phone, email)
  • Precise location data (GPS coordinates)
  • Device identifiers (IDFA, advertising ID)
  • Usage analytics data
  • Crash reports
  • Any information that can be used to track you

2. Permissions

GhostNote may request the following system permissions, all used for local features only:

Permission Purpose Required
Microphone Record voice notes Required (core feature)
Speech Recognition Convert speech to text (local processing) Required (core feature)
Location Services Add city-level geographic context Optional
Calendar Add reminders to system calendar Optional
Media Library Record currently playing music name Optional

Important: All optional permissions can be denied without affecting the core note-taking functionality.

3. Data Storage and Security

3.1 Local Storage

  • All data is stored only on your iOS device locally
  • SQLite database is used for note content and metadata
  • Audio files are stored in the app sandbox Documents directory
  • No use of iCloud or any cloud services for data sync

3.2 Data Security

  • Data is protected by iOS system-level encryption
  • App sandbox mechanism ensures other apps cannot access your data
  • No data sharing with any third parties
  • No user data transmitted over the network

4. Third-Party Services

GhostNote does NOT use any third-party services, including but not limited to:

  • Analytics services (e.g., Google Analytics, Firebase Analytics)
  • Ad networks
  • Crash reporting services
  • Cloud storage services
  • Social media SDKs

The only external library used is the open-source GRDB.swift, solely for local database management, with no network communication involved.

5. AI Features and Privacy

GhostNote uses the following AI features, all processed locally on device:

  • Speech Recognition: Uses Apple Speech Recognition API, completely offline
  • Natural Language Processing: Uses Apple Natural Language framework for intent classification and sentiment analysis
  • Smart Categorization: Uses local machine learning models to identify note types (reminders, todos, ideas)
  • Semantic Search: Uses Apple NLEmbedding to generate semantic vectors for local similarity search

Privacy Guarantee: Your voice and text content is never sent to any remote servers for processing.

6. Data Retention and Deletion

  • You can delete any note within the app at any time
  • Deleted data is permanently removed from the local database
  • Uninstalling the app will delete all app data, including notes, recordings, and settings
  • We do not retain any backup copies of your data

7. Children's Privacy

GhostNote is suitable for users of all ages. Since we do not collect any personal information, we do not knowingly collect personal data from children under 13.

8. Updates to Privacy Policy

We may update this privacy policy from time to time. When updated, we will revise the "Last Updated" date at the top of the page. Significant changes will be notified within the app.

9. Your Rights

Since all data is stored only on your device, you have complete control:

  • Right to Access: You can view all your note data at any time
  • Right to Modify: You can edit or delete notes at any time
  • Right to Delete: You can delete individual notes or uninstall the app to remove all data
  • Right to Revoke: You can revoke granted permissions anytime in iOS Settings
